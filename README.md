## 📋 Sobre o projeto

Este é um projeto base que utiliza uma arquitetura moderna de desenvolvimento full-stack, com separação entre frontend, backend e banco de dados. O projeto demonstra uma aplicação completa de gestão de dados com autenticação, CRUD e upload de arquivos.
